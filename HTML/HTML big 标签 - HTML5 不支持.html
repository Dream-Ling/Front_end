HTML <big> 标签 - HTML5 不支持

创建时间：2018/8/22 19:35
作者：凌小康

实例

让文本比常规的字体大一号：
<p><big>这个文本比较大。</big></p>

尝试一下 »

浏览器支持

    
所有主流浏览器都支持 <big> 标签。

标签定义及使用说明

HTML5 不支持 <big> 标签。请用 CSS 代替。
<big> 标签用来制作更大的文本。

提示和注释

提示：在文档中使用 CSS 来规定文本大小。

HTML 4.01 与 HTML5之间的差异

HTML5 不支持 <big> 标签，HTML 4.01 支持 <big> 标签。
