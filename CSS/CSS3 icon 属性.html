CSS3 icon 属性

创建时间：2018/8/31 15:34
作者：凌小康

实例

将图像元素设置为图标化的等价物：
img
{
content:icon;
icon:url(imgicon.png);
}

浏览器支持

没有主流浏览器支持icon属性。

属性定义及使用说明

icon 属性为创作者提供了将元素设置为图标等价物的能力。
注意：元素的图标是不可使用，除非"content"属性设置为"icon"！
默认值：
auto
继承：
no
版本：
CSS3
JavaScript 语法：
object.style.icon="url(image.png)"

语法

icon: auto|URL|inherit;

值
描述
auto
使用由浏览器提供的默认通用图标。
URL
引用列表中的一个或多个图标，列表用逗号分隔。
inherit
规定应从元素继承 icon 属性的值。
