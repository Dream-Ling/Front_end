CSS background-blend-mode 属性

创建时间：2018/8/30 16:28
作者：凌小康

实例

混合模式:
div { 
    width: 290px;
    height: 69px;
    background-size: 290px 69px;
    background-repeat:no-repeat;
    background-image: linear-gradient(to right, green 0%, white 100%), url('logo.png');
    background-blend-mode: color-dodge;
}

尝试一下 »

定义和用法

background-blend-mode 属性定义了背景层的混合模式（图片与颜色）。
默认值:
normal
Inherited:
no
Animatable:
no. 查看 animatable
版本:
CSS3
JavaScript 语法:
object.style.backgroundBlendMode="screen"

CSS 语法

background-blend-mode: normal|multiply|screen|overlay|darken|lighten|color-dodge|saturation|color|luminosity;
