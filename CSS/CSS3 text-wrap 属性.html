浏览器支持

任何主流浏览器都不支持text-wrap属性。

属性定义及使用说明

text-wrap属性指定文本换行规则。
默认值：
normal
继承：
yes
版本：
CSS3
JavaScript 语法：
object.style.textWrap="none"

语法

text-wrap: normal|none|unrestricted|suppress;

值
描述
normal
只在允许的换行点进行换行。
none
不换行。元素无法容纳的文本会溢出。
unrestricted
在任意两个字符间换行。
suppress
压缩元素中的换行。浏览器只在行中没有其他有效换行点时进行换行。
