CSS3 的文本阴影

CSS3 中，text-shadow属性适用于文本阴影。
您指定了水平阴影，垂直阴影，模糊的距离，以及阴影的颜色：

实例

给标题添加阴影:
h1 { text-shadow: 5px 5px 5px #FF0000; }

CSS3 中 CSS3 box-shadow 属性适用于盒子阴影

实例

实例
