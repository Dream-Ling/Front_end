属性定义及使用说明

width属性设置元素的宽度。
注意： width属性不包括填充，边框和页边距！
默认值：
auto
继承：
no
版本：
CSS1
JavaScript 语法：
object.style.width="50px"

属性值

值
描述
auto
默认值。浏览器可计算出实际的宽度。
length
使用 px、cm 等单位定义宽度。
%
定义基于包含块（父元素）宽度的百分比宽度。
inherit
规定应该从父元素继承 width 属性的值。
