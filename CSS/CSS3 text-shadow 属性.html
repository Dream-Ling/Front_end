属性定义及使用说明

text-shadow 属性应用于阴影文本。
默认值：
none
继承：
yes
版本：
CSS3
JavaScript 语法：
object.style.textShadow="2px 2px #ff0000"

语法

text-shadow: h-shadow v-shadow blur color;
注意： text-shadow属性连接一个或更多的阴影文本。属性是阴影，指定的每2或3个长度值和一个可选的颜色值用逗号分隔开来。已失时效的长度为0。
值
描述
h-shadow
必需。水平阴影的位置。允许负值。
v-shadow
必需。垂直阴影的位置。允许负值。
blur
可选。模糊的距离。
color
可选。阴影的颜色。参阅 CSS 颜色值。
