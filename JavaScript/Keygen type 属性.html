Keygen type 属性

作者：凌小康

 Keygen 对象

实例

返回form元素keygen字段的类型：
var x = document.getElementById("myKeygen").type;
x 输出结果:
keygen

尝试一下 »

定义与使用

type 属性返回form元素的field字段类型。
对于 keygen 字段,属性总是返回 "keygen".
注意: 该属性是只读的。

浏览器支持

    
所有主流浏览器都支持 disabled 属性，除了Internet Explorer。

语法

keygenObject.type

技术细节

返回值:
字符串, 代表着form元素keygen字段的类型。
