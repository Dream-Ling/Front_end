Columngroup 对象

Columngroup 对象代表着HTML表格（table）的一组列。
在HTML文档中每个 <colgroup> 标签，都会创建一个 Columngroup 对象。
如需对全部列应用样式，<colgroup> 标签很有用，这样就不需要对各个单元和各行重复应用样式了。<colgroup> 标签只能在 table 元素中使用。

Columngroup 对象属性

属性
描述
span
设置或者返回列组中span属性的值。

标准属性和事件

Columngroup 对象同样支持标准 属性 和 事件。
