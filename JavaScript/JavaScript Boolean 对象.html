Boolean 对象

Boolean 对象用于转换一个不是 Boolean 类型的值转换为 Boolean 类型值 (true 或者false).
Boolean 对象完整教程, 请访问 JavaScript Boolean 对象教程。

Boolean 对象属性

属性
描述
constructor
返回对创建此对象的 Boolean 函数的引用
prototype
使您有能力向对象添加属性和方法。

Boolean 对象方法

方法
描述
toString()
把布尔值转换为字符串，并返回结果。
valueOf()
返回 Boolean 对象的原始值。
